Offsite XML Design Pattern: Footnote
This page reviews an XML design pattern that is located at an external website.

Abstract:

From the pattern: "The Footnote Pattern is used when running text refers to an external note or annotation that is displayed in a separate flow and linked by a symbol such as a number."


Discussion:

This pattern is from The XML Specification Guide Web site by Ian Graham and Liam Quin.

This is on of a group of documentation oriented design patterns presented at the site.



Related Patterns:

The Referenced Note Pattern is a generalization of the Footnote Pattern.
